Entendendo a comunicação Client x Server

>>Voltar

Neste módulo aprendi a diferença entre o cliente, no qual consiste em fazer requisições de serviços ou informações de outro dispositivo (servidor) e, o servidor que consiste em fornecer essas informações para os clientes.Em resumo, o cliente é quem solicita e o servidor é quem fornece os recursos ou serviços solicitados pelo cliente. Essa distinção é fundamental em arquiteturas de rede cliente-servidor, onde os clientes dependem dos servidores para obter os recursos necessários para realizar suas tarefas.

Introdução ao Html na prática

>>Voltar

Este módulo apresentará na prática como é um documento HTML e o que ele precisa para realmente ser um documento HTML.Além disso o módulo contará com a explicação de tags e suas respectivas funções de modo a melhorar a compreensão do desenvolvedor.

Estrutura do HTML e funções das tags.

>>Voltar

Esse módulo começa com a explicação da estrutura básica do HTML5 que consiste em 

                 
                    
                        < !DOCTYPE html>
                        < html lang="pt-br"> 
                        < head>
                        < meta charset="UTF-8">
                        < meta name="viewport" content="width=device-width, initial-scale=1.0">
                        < title>Exibe o título da página< /title>
                        < /head>
                        < body> < /body>
                        < /html>


A tag< !DOCTYPE html> mostra que é um HTML5 a versão mais recente do documento.

A tag< html> < /html> mostra que é um documento html, então tudo que estiver dentro dessa tag vai ser um Doc. escrito em HTML.

A tag < head> < /head> consiste na cabeça do documeto, essa tag exibe informações que não são visíveis para o usuário.

Corpo do documento e tags de título e parágrafos

>>Voltar

A tag < body> < /body> é o corpo do documento, todo o conteúdo que é exibido para o usuário fica dentro dessa tag, aqui podem ser inseridos elementos como textos, imagens,links, listas, formulários. Para inserir um texto que tenha um título ultiliza-se as tags do h1 ao h6 que geralmente possuem parágrafos e vão ser utilizadas na seguinte sequência:

< h1> < /h1>Exibe o título principal da página (título mais importante na página). Deve ser utilizado apenas uma vez por página.

< h2> < /h2>São os títulos de segundo nível (usado para exibir subseções/subtítulos importantes da página). Pode ter vários em uma página desde que seja usado de forma lógica.

< h3> < /h3>Utilizado para subdividir seções dentro dos h2s(Organizam o conteúdo em seções menores e mais específicas)

< h4> < /h4>Muito parecído com o h3, porém é utilizado para subseções dentro do h3 de forma cada vez mais específica.

< h5> < /h5>Utilizada para subseções dentro do h4 (utilizada com menos frequência).

< h6> < /h6>É o nível mais baixo de título (É também o menos comum e é utilizado para subseções muito específicas dentro do h5)

Ainda dentro da tag < body> < /body> é possível inserir uma tag que cria parágrafos, denominada como < p> < /p>. Essa tag pode ser usada abaixo de uma das tags de títulos depois da abertura e fechamento.

Exemplo: 
            
                
                    
                        < h1>Forma correta de utilizar a tag p < /h1>
                        < p>Isso é um parágrafo < /p>

Listas ordenadas e não ordenadas

>>Voltar

Dentro do HTML é possível utilizar listas ordenadas < ol> < /ol> (São usadas para criar listas com uma ordem numérica ou alfabética) e não ordenadas < ul> < /ul> (Cria listas onde a ordem dos itens não é importante. Cada item é marcado com um marcador padrão (geralmente um ponto)).

Exemplo de listas ordenadas:

Comando da lista: 



                
                    
                        < ol>
                            < li>Item 1< /li>
                            < li>Item 2< /li>
                            < li>Item 3< /li>
                          < /ol>
                    
Exemplo de como fica:
                     
    
                        
  1. Item 1
    2. 
                        
  2. Item 2
    3. 
                        
  3. Item 3
    4.
Exemplo de listas não ordenadas:

Comando da lista:

                
                    
                        < ul>
                            < li>Item 1< /li>
                            < li>Item 2< /li>
                            < li>Item 3< /li>
                          < /ul>
                    
Exemplo de como fica:
                     
    
                        
  • Item 1
    • 
                        
  • Item 2
    • 
                        
  • Item 3
    •


Tags "a", "hr", "sub", "sup", "blockquote", "font", "dell" e "abbr".

>>Voltar

< a >: Esta tag é usada para criar links em HTML.Exemplo do código abaixo:

                < a href="https://www.dio.me/">Visite dio.com< /a>
Agora clique no link do exemplo: Visite dio.com


< hr>: Essa tag é usada para criar uma linha horizontal, muitas vezes usada para separar conteúdos.
Exemplo: 

                
                    
                        < p>Conteúdo acima< /p>
                        < hr>
                        < p>Conteúdo abaixo< /p>
                        < /code>


Agora a demonstração de como ficaria:

Conteúdo acima

Conteúdo abaixo


< sub> Essa tag é usada para criar texto subscrito, que aparece menor e abaixo da linha de base normal do texto. Por exemplo, para expressar fórmulas químicas: 

                
                    H< sub>2< /sub>O
Agora a demonstração de como ficaria:
H2O


< sup >: Essa tag é usada para criar texto sobrescrito, que aparece menor e acima da linha de base normal do texto. Por exemplo, para expressar expoentes: 

                    
                        x< sup>2< /sup>

                        *Lembre-se de que não precisa dar esses espaços nas tags,só está assim para aparecer o exemplo do código na página

Agora a demonstração de como ficaria:
x2


< blockquote >: Essa tag é usada para criar citações longas em um bloco de texto. Por exemplo: 

                
                    
                        < blockquote >
                            < p >Esta é uma citação longa. Ela pode se estender por vários parágrafos.< /p >
                            < p > - Autor da Citação< /p >
                          < /blockquote >
Agora a demonstração de como ficaria:

Esta é uma citação longa. Ela pode se estender por vários parágrafos.

- Autor da Citação


< font >: Esta tag é usada para definir as características da fonte para o texto contido nela. No entanto, é considerada obsoleta e não é recomendada para uso no desenvolvimento web moderno.


< del >: Esta tag é usada para indicar texto deletado ou removido dentro de um documento.

Demonstração do código:

                
                    < p >Eu removi < del >algum texto< /del > deste parágrafo.< /p >
Como fica:

Eu removi algum texto deste parágrafo.


< abbr >: Esta tag é usada para definir uma abreviação ou um acrônimo. Geralmente é usada para fornecer uma explicação ou expansão do texto abreviado, que pode ser exibida como uma dica de ferramenta quando o usuário passa o mouse sobre a abreviação.
Demonstração do código: 

                
                    < p >< abbr title="World Health Organization">OMS< /abbr > é uma organização internacional de saúde.< /p >
Como ficaria:

OMS é uma organização internacional de saúde.
